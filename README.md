# FallTrap

<div align="center">
  <strong>Advanced FallTrap & Base Generator for PocketMine-MP 5</strong>
  <br>
  <em>Create custom falltraps with a simple command and an automatic NPC.</em>
</div>


## Features

- **Auto NPC** — An NPC spawns automatically with your skin holding a pickaxe.
- **Hologram** — The NPC shows a floating hologram with info.
- **One Command** — Just `/falltrap` to open the menu and generate traps instantly.
- **Customizable** — Change trap size, depth, and material.
- **Permission-Based** — Control who can use the command..

## How It Works

1. Type `/falltrap`
2. An NPC appears automatically with **your skin** holding a **diamond pickaxe**.
3. A hologram floats above the NPC with information.
4. Click the NPC to open the GUI and create your falltrap.

## Command

| Command | Description | Permission |
|---|---|---|
| `/falltrap` | Open the FallTrap menu | `falltrap.command` |

## Permission

| Permission | Description | Default |
|---|---|---|
| `falltrap.command` | Allows player to use /falltrap | `op` |
